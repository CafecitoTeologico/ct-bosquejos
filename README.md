# ct-bosquejos
Colección de bosquejos, a modo de repositorio personal pero también puestos a disposición del estudio y uso de quien lo necesite.

# Uso
Se pone a disposición del público para su uso libre, bajo licencia AGPLv3, sin afán limitar el uso al mismo tiempo que se mantienen valores esenciales como la importancia de reconocer/atribuir la autoría y compartir con la misma licencia lo que se ha podido disfrutar de manera libre.
